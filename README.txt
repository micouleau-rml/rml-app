RML PLIAGE ALU — package GitHub Pages

Mise en ligne rapide :
1. Créez un dépôt GitHub ou ouvrez celui qui héberge déjà votre application.
2. Déposez ces fichiers à la racine du dépôt, ou dans le dossier /docs.
3. Dans GitHub :
   - Settings
   - Pages
   - Build and deployment
   - Deploy from a branch
   - Branche : main
   - Dossier : /root ou /docs
4. Attendez la publication du site.
5. Ouvrez le lien github.io sur iPhone dans Safari.
6. Faites Partager → Sur l’écran d’accueil.

Fichiers inclus :
- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png
- apple-touch-icon.png
